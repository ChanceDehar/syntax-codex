# Syntax Codex

A desktop application for storing and searching programming cheat sheets with cloud synchronization.

## Features

* **Cloud-Synced Storage**: All cheat sheets saved to Firebase Firestore
* **Smart Search**: Real-time filtering by name, description, or content
* **Multi-Language Support**: Filter by 15+ programming languages
* **Quick Copy**: One-click clipboard copy of code snippets

## Tech Stack

* **Frontend**: JavaFX 21
* **Backend**: Firebase Authentication, Firestore
* **Build Tool**: Maven
* **Language**: Java 17
